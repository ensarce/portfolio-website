# Yazılım Geliştirici Portföyü

Bu proje, C#, .NET, Angular ve Sağlık Sistemleri alanında uzmanlaşmış bir yazılım geliştiricisinin portföyüdür.

## Özellikler

- Modern ve duyarlı tasarım
- Karanlık tema ile glassmorphism efektleri
- Animasyonlar ve geçiş efektleri
- SEO dostu
- PWA (Progressive Web App) desteği
- Çoklu dil desteği (Türkçe)

## Teknolojiler

- Angular 17+ (Standalone Components)
- TailwindCSS
- TypeScript
- RxJS
- HTML5 & SCSS

## Kurulum

1. Bağımlılıkları yükleyin:
   ```
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```
   ng serve
   ```

3. Tarayıcınızda `http://localhost:4200` adresine gidin.

## Yapılandırma

Proje iki ortamda çalışacak şekilde yapılandırılmıştır:

- Geliştirme: `src/environments/environment.ts`
- Üretim: `src/environments/environment.prod.ts`

## Dağıtım

Projeyi üretim için derlemek için:

```
ng build --prod
```

Derlenen dosyalar `dist/` klasörüne yerleştirilecektir.
